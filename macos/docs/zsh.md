# ZSH

## .zshrc

if files exist, remove them

```bash
rm ~/.zshrc
```

then create symlinks

```bash
ln -s ~/dotfiles/macos/data/.zshrc ~/.zshrc
```

## Powerlevel10k

if files exist, remove them

```bash
rm ~/.p10k.zsh
```

then create symlinks

```bash
ln -s ~/dotfiles/global/.p10k.zsh ~/.p10k.zsh
```
